# Time Series Vision Trading

Time series trading analysis using computer vision and deep learning.

## Features

- Converts time series trading data into visual representations
- Uses computer vision models for pattern recognition
- Technical analysis integration
- Stock data fetching via yfinance

## Installation

```bash
pip install -r requirements.txt
```

## Technologies

- PyTorch and TorchVision for deep learning
- yfinance for market data
- Technical Analysis library (ta)
- Pandas and NumPy for data processing
- Matplotlib for visualization
